Gemini CLI
奇幻旅程

一步一步，將你的終端機打造成最強大的 AI 工作坊。專為初學者設計的互動式指南。

01

前置作業

在開始之前，我們需要先在你的電腦中安裝兩位強大的助手：Node.js 與 Git。

穩定環境 1. 安裝 Node.js 版本控制 2. 安裝 Git
02

安裝 Gemini CLI

萬事俱備！現在讓我們透過終端機（命令提示字元）來召喚 Gemini CLI。

  • 在電腦中搜尋並打開「命令提示字元 (CMD)」或「PowerShell」。
  • 輸入以下指令並按下 Enter：
npm install -g @google/gemini-cli
> npm install -g @google/gemini-cli
added 592 packages in 36s
成功！你的 AI 助手已就位。
03

啟動你的 AI 助手

現在，我們要把 Gemini 帶到你的工作目錄中。

  • 到檔案總管，複製你想要進行專案的資料夾路徑。
  • 在終端機輸入 cd 加上空格，然後貼上路徑：
cd C:\你的\專案\路徑

接著，輸入以下指令正式啟動：

gemini
04

初試啼聲

Gemini CLI 不僅擁有強大的大腦，還能直接操控你的電腦文件。

💡 差異在於：
1. 他能直接在你的資料夾生成檔案。
2. 他擅長將複雜任務拆解成多個步驟。

試試看這樣下指令：

請先寫一份「飼養老虎在太空站競速對戰的末日科幻派對遊戲」企劃書。寫完後，另外做一個互動式網頁來解說這個企劃。
05

安裝延伸功能 & Skill

想讓 Gemini 更強大嗎？安裝延伸功能是必經之路。請先輸入 exit 退出 gemini 模式。

1. Google 雲端全家桶 (Workspace)

gemini extensions install https://github.com/gemini-cli-extensions/workspace

2. 前端設計 Skill (Frontend Design)

gemini extensions install https://github.com/buildatscale-tv/gemini-skills
06

終極驗證

現在，重新啟動 gemini 並再次執行剛才的「太空老虎遊戲」指令。你會發現，有了前端 Skill 的幫助，生成的網頁將會比剛才更漂亮、更專業！

恭喜！你已經掌握了 Gemini CLI 的魔法。
